# Getting started

Requirements:
- python 3
- pip

Python modules to install:
- pygame
- ttkbootstrap
- pyinstaller

Running the game in dev mode:
```sh
python Chess.py
```

Building the game locally:
```sh
pyinstaller --onefile Game.py --name Mirror_Chress_1_2
# run with
./dist/Mirror_Chress_1.2
```
